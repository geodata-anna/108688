## Introduction to Pandas DataFrames – Replicating an article in Markdown

This project contains the following key files:

- **Pandas.ipynb**: Das Jupyter-Notebook, welchen den Pandas-Artikel nachbildet.  
  Enthält Markdown-Erklärungen und ausführbaren Python-Code mit Outputs.
- **Pandas.html**: Der erstellte HTML-Artikel aus dem Jupyter-Notebook.
- **Pandas.webp**: Das vorgegebene Bild, welches im Artikel und Notebook verwendet wird..
